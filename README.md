
# 🚀 GrowEasy AI CSV Importer

An intelligent CSV importer that uses AI to map any CSV format to GrowEasy CRM format.

## ✨ Features

- 📤 Drag & drop CSV upload
- 📊 Beautiful data preview with sticky headers
- 🤖 AI-powered intelligent field mapping
- 📈 Batch processing for large files
- 📊 Import statistics and error handling
- 🎨 Modern responsive UI

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- OpenAI API Key
